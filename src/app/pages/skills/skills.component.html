HTML5

HTML (HyperText Markup Language) is the standard language used to create web pages and web applications.

CSS3

CSS3 is the latest evolution of CSS, used for styling and designing web pages.

JavaScript

JavaScript is a scripting language used to create interactive and dynamic web pages.

Angular

Angular is a TypeScript-based front-end framework for building single-page applications (SPAs).

Python

Python is a high-level, interpreted programming language known for its simplicity and readability.

Django

Django is a high-level Python web framework that helps build secure, scalable, and maintainable web applications.

Pandas

Pandas is a powerful Python library used for data analysis, manipulation, and cleaning.

NumPy

NumPy (Numerical Python) is a powerful Python library used for numerical computing.

MySQL

MySQL is an open-source relational database used to store, manage, and retrieve data efficiently.

MongoDB

MongoDB is a document-oriented NoSQL database that stores data in JSON-like BSON format.

Netlify

Netlify is a cloud-based platform for deploying and hosting static websites and serverless applications.

Git & GitHub

Git is a distributed version control system that helps developers track changes and collaborate.
